Small Visual Basic (sVB), is an educational programming language evolved from Microsoft Small Basic, designed to be even easier and more powerful for kids and young beginners. It added the missing Small Basic features like variable domains, functions, subroutine parameters, and objects. sVB also integrates some modern syntax features from VB .NET and uses a  VB6-Like simple form designer and event handlers.
 
Why Choose sVB Over Python?

  • Focused Purpose: sVB is tailored for educational purposes and desktop apps.
  • Path to DotNET: sVB acts as a stepping stone to VB .NET and C#.
  • Advanced Topics: sVB allows multi-form apps, XAML styles and multi-threading, and creates code libraries.
  • Dynamic Typing: Easier without needing type conversions.
  • Case Insensitivity: More user-friendly for kids.
  • Natural Array Indexing: 1-based indexes by default, but flexible.
  • Degree Angles Option: Use degrees in trigonometry functions if you want, which is easier for kids.
  • Simpler For Loops: Intuitive looping syntax without the confusion or the range exclusive end.
  • Readable Code Structure: End blocks and automatic formatting saves the kid from the confusing Python block indentation.
  • Powerful IDE: Easy handling of forms and graphics.


Target Audience:

  • Kids and teens (6-18 years): Tailored lessons matching their educational level.
  • Beginners: Parents and teachers can guide kids with step-by-step exercises.
  • Small Basic Developers: Transition from SB to sVB with improved features and Windows Forms.
  • Other BASIC Dialects: Easier transition to VB.NET for users of QBasic, VB6, VBA, or VBScript.


What will you learn in this series:
Level1:

  • Download and install sVB.
  • Use the form designer.
  • Use the Text property of the TextBox control.
  • Handle the OnClick event of the button control.
  • Write a simple 1-line code.
  • Concatenate strings.
  • Use simple arithmetic operations.
  • Crate a calculator with a simple user interface.

Level2:

  • Use the text window to write and read data.
  • Call methods and pass arguments to them.
  • Use variables to store date and read it back.
  • Show a message box.
  • Get the current date and time.
  • Use the timer to crate a clock.

Level3:

  • Use the turtle to draw and paint figures.
  • Draw squares and rectangles.
  • Calculate the area and perimeter of the square, rectangle and circle.
  • Round a decimal number to the nearest integer.
  • Draw triangles and quadrilaterals.
  • Defining Subroutines and parameters.
  • Using for loops.
